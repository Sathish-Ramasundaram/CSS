The CSS class Selector

The class selector selects HTML elements with a specific class attribute.

1. To select elements with a specific class, write a period (.) character, followed by the class name. Click here to see the Example
2. You can also specify that only specific HTML elements should be affected by a class. Click here to see the Example
3. HTML elements can also refer to more than one class. Click here to see the Example

Note: A class name cannot start with a number!
