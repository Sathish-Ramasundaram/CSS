The CSS element Selector

The element selector selects HTML elements based on the element name.

Example:


        <!DOCTYPE html>
        <html>
        <head>
        <style>        
        
            p{
            text-align: center;
            color: red;
            } 
        
        </style>
        </head>
        <body>
        <p>Every paragraph will be affected by the style.</p>
        <p id="para1">Me too!</p>
        <p>And me!</p>
        </body>
        </html>
